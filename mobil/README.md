# UrbanChain Mobile

Flutter mobil uygulama iskeleti.

## Ekranlar

- Splash
- Giris
- Ana sayfa
- Rapor olustur
- AI analizi
- Gonderildi
- Raporlarim
- Rapor detayi
- Bildirimler
- Profil

## Calistirma

```bash
cd mobil
flutter pub get
flutter run
```

## Web destegi

Bu proje Flutter Web ile de calisacak sekilde hazirlandi.

- Yerelde web icin `flutter run -d chrome` kullan.
- Backend farkli bir adreste ise `URBANCHAIN_API_BASE_URL` tanimla.
